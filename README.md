## Installation

**To run Locally**

Install all dependencies:
`npm install`

Run build from production server:
`node index.js`

**To develop react app**

Install all dependencies:
`npm install`

Start local development server:
`npm start`

Build any changes into production ready files used by express server:
`npm run build`
